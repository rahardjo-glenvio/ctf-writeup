# picoCTF Challenge Writeups

Personal collection of CTF solutions from picoCTF platform.

##  Repository Structure
```
picoctf/
├── Cryptography/           # Crypto challenges and solutions
├── Reverse Engineering/    # Binary and bytecode analysis
└── Web Exploitation/       # Web security challenges
```

##  Focus Areas

This repository currently focuses on three main categories:

- **Cryptography** - Classical ciphers, RSA vulnerabilities, steganography
- **Reverse Engineering** - Binary analysis, bytecode reversing, program flow
- **Web Exploitation** - SQL injection, XSS, SSRF, authentication bypass

##  What's Inside

Each challenge includes:
- Original challenge files
- Solver/exploit scripts
- Detailed writeups with step-by-step solutions
